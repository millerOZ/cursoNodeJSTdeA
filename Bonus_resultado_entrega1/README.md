## Pasos para ejecutar
* Descargar el proyecto de GitHub
* npm install
* Ejecute: "node principal inscribir" para inscribirse un curso

--id, -i                                                            [required]

--nombre, -n                                                        [required]

--cedula, -c                                                        [required]

* id de cursos 1,2,3
* Ingrese en el navegador de preferencia **http://localhost:3000/**
